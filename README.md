fork自weiyigulu，删掉了其他协议，保留CDT，增加了CDT的遥脉



CDT依赖引入

```bash
<dependency>
  <groupId>wei.yigulu</groupId>
  <artifactId>protocol-cdt</artifactId>
  <version>1.0.0</version>
</dependency>
```
